# Sleep Health and Lifestyle Analysis

This project explores a sleep health and lifestyle dataset using Python in Jupyter Notebook. It focuses on understanding how factors such as sleep duration, quality of sleep, stress level, physical activity, BMI category, heart rate, blood pressure, and daily steps relate to sleep disorders.

## Project Overview

The aim of this project is to perform basic data analysis on sleep and lifestyle data to identify patterns and relationships between health indicators and sleep disorders such as insomnia and sleep apnea.

This project is useful for practicing:

- data loading and inspection
- filtering and selecting data
- grouping and aggregation
- health and lifestyle data analysis
- Jupyter Notebook workflow using Python
